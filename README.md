# falling sand sim

this is a fun little sand simulation that i built to run entirely in the windows console using my custom [rusty_console_game_engine](https://github.com/rip-super/https://github.com/rip-super/RustyConsoleGameEngine)!!

### notes
this only works on windows 10 or 11. make sure to use conhost.exe to run the application, otherwise the console rendering is not gonna be correct

### controls
* left click: use current tool
* 1: select sand tool
* 2: select delete tool
* c: clear the screen
* up/down arrows: change brush size
